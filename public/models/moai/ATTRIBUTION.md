# Moai 3D Asset Attribution

This project can load an external Moai model from:
`/public/models/moai/angelito-moai.glb`

## Selected model
- Title: Angelito-Moai (Hamburg)

## How to install locally
1. Log into Sketchfab and download the model as `.glb`.
2. Rename the file to `angelito-moai.glb`.
3. Place it in this folder: `/public/models/moai/`.

If this file is missing, the app automatically falls back to the built-in procedural Moai.
